Nachrichten an Mia werden über Supabase Edge Functions an die Anthropic API weitergeleitet. Deine Nachrichten werden nicht zum KI-Training verwendet (API-Nutzung, siehe anthropic.com/privacy). Sende keine sensiblen persönlichen Daten im Chat.

TinyMinds enthält keine Werbung, kein Analytics-Tracking, keine Cookies (außer Auth), kein Fingerprinting und keine Datenweitergabe an Dritte zu Werbezwecken.
